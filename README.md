# Frontend Dashboard – AG Grid (React + Vite)

This project is a client-side dashboard built using **React**, **Vite**, and **AG Grid**.  
It demonstrates a clean and scalable approach to rendering tabular data with sorting, filtering, pagination, and subtle UI enhancements.

The focus of this assignment was to keep the implementation **simple, readable, and production-oriented**, while following AG Grid best practices.

---

## Tech Stack

- React
- Vite
- AG Grid (Community Edition)
- JavaScript
- CSS

---

## Key Features

- Client-side AG Grid implementation
- Sortable and filterable columns
- Global search across all columns
- Pagination with default page size set to 10
- Custom role indicator using subtle vertical color accents
- Role legend displayed alongside the search bar
- Clean and professional dashboard layout

---

## Application Overview

The dashboard displays tabular employee data using AG Grid’s client-side row model.  
The grid is configured to handle sorting, filtering, and pagination efficiently while keeping the UI responsive and easy to use.

Visual enhancements such as role indicators and the legend are intentionally minimal to maintain clarity and avoid unnecessary visual noise.
